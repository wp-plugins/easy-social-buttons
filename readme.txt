It is very easy and simple Plugin for showing your social links using a simple widget so that people can connect with you more easily.

== Description ==

When you activate this plugin widget, you'll see 'Easy Social Buttons' appear in the widget's list. Just drag and drop it wherever you wish to show your social links on the website.

Within the widget, you can add a 'Title' that will appear on the website as widget's title. Then there is a list of social websites with a space for adding your profile URLs for each respective social site. Only those social icons will be displayed on your website for which you have added a URL.

For example - If you provide a URL in the text box of Twitter, Pinterest and you leave the text box of Facebook blank. Then, you'll only see Twitter, Pinterest icons appear in the widget on the website. So its easily customizable and you can 'Choose what you want to Share'.

== Installation ==

1. When you download this plugin widget it will mainly be a folder named 'easy-social-buttons'.
2. 'easy-social-buttons' folder contains 3 items: 1) index.php file 2) assets folder 3) readme.txt
3. Upload 'easy-social-buttons' folder to the `/wp-content/plugins/` directory.
4. Activate this 'Easy Social Buttons' plugin widget through the 'Plugins' menu in WordPress.
5. After activating you will see the 'Easy Social Buttons' widget appear in the widgets list.
6. Customize the widget as per your need and see a cool social widget appear on your website.

== Features ==

1. At present, There are 10 social websites which have been included viz. Facebook, Google+, Instagram, MyWorld, Odnoklassniki, Pinterest, Tumblr, Twitter, VK, YouTube.
2. You can change the color of icons for your website design. The default is color #333 for a standard Wordpress theme - "Twenty Fifteen". It is recommended to use the color of the text block in which you place the widget. Background transparent icons.
3. There is a steep coloring effect icons in the social network brand color when the mouse cursor.
4. Adaptive design allows the use of plug-in sites with the technology «Responsive Web Design».
5. Using technology svg, it allows the display icons is perfect on any device screen and browser. This plugin is not being used is not a picture.
6. The plug is not used does not have Java Scripts.
7. Designed in strict accordance with the requirements of trademark owners to third parties upstream developer. 100% of the color of these brands.
Note: Be sure to enter the URL for each social site by starting with 'http://' or 'https://' (Example: http://www.facebook.com/beauty) in order to make the social icons properly link to your social profile.
